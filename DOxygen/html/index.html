Shapes Project

Program Introduction

The Shapes application is meant to show how inheritance and overloaded operatores work and use templates and exceptions.

Project Requirements

This program asks to create the Shape class when it's the parent class, also it asked to create two more classes (Circle and Square) as the childerns of shape class with inheritance. The program asks to use overloaded operators for the circle shape and square. The program attends to use templates and exceptions
